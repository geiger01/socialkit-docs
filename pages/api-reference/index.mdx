# API Reference Overview

Explore the following API documentation sections for detailed information on each endpoint:

> **Note:** You can make requests using either GET or POST methods.

## Interactive Playground

Prefer a "try it now" experience? **[Open the interactive API playground →](https://api.socialkit.dev/docs)**

Every endpoint listed below is available there with live request builders, response previews, and a downloadable OpenAPI 3.0 spec at [`api.socialkit.dev/openapi.json`](https://api.socialkit.dev/openapi.json) (import into Postman, Insomnia, or generate clients with `openapi-generator`).
