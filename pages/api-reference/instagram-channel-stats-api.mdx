# Instagram Channel Stats API

The Instagram Channel Stats API retrieves comprehensive profile statistics and metadata for Instagram accounts, including follower counts, post counts, bio information, and verification status.

## Endpoint

```bash copy
https://api.socialkit.dev/instagram/channel-stats
```

#### Example Request

```bash copy
GET https://api.socialkit.dev/instagram/channel-stats?access_key=<your-access-key>&url=https://instagram.com/yuumi_cat9
```

## Parameters

**`url`** <small style={{margin:'0 10px'}}>string</small> <small style={{color: 'red'}}>Required</small><br/>
The Instagram profile URL of the account to get statistics for. Supports various Instagram URL formats.

---

**`access_key`** <small style={{margin:'0 10px'}}>string</small> <small style={{color: 'red'}}>Required</small><br/>
Your API access key. Can be provided via the `access_key` query parameter, `x-access-key` header, or request body.

---

**`cache`** <small style={{margin:'0 10px'}}>boolean</small> <small style={{margin:'0  10px  0 0'}}>Optional</small> <small>Defaults to false</small><br/>
Cache the response for faster subsequent requests.

---

**`cache_ttl`** <small style={{margin:'0 10px'}}>number</small> <small style={{margin:'0  10px  0 0'}}>Optional</small> <small>Defaults to 2592000</small><br/>
Cache the response for a custom TTL (in seconds). Maximum `2592000` seconds (1 month), minimum `3600` seconds (1 hour).

## Response Fields

The API returns the following data fields:

- **`profileUrl`**: The Instagram profile URL
- **`username`**: The account's username (without @ prefix)
- **`nickname`**: The account's display name
- **`verified`**: Whether the account is verified (boolean)
- **`followers`**: Total follower count (number)
- **`following`**: Total following count (number)
- **`totalPosts`**: Total number of posts published (number)
- **`bio`**: The account's bio/description text
- **`avatar`**: URL to the profile avatar image

## Use Cases

- **Influencer Research**: Analyze Instagram creator profiles and audience size for partnerships
- **Competitive Analysis**: Monitor competitor account growth and engagement metrics
- **Brand Monitoring**: Track brand presence and performance on Instagram
- **Creator Discovery**: Find and evaluate potential content creators for collaborations
- **Analytics Dashboards**: Build comprehensive Instagram analytics and reporting tools
- **Market Research**: Understand account demographics and popularity trends
- **Performance Tracking**: Monitor your own or client Instagram account statistics over time
- **Verification Status**: Identify verified accounts for partnership opportunities
- **Audience Analysis**: Compare follower counts and engagement across multiple accounts

## Notes

- Statistics are retrieved in real-time from Instagram
- All numeric fields are returned as integers
- Avatar URLs are temporary and may expire after some time
- Follower, following, and post counts reflect current values
- No need to extract account ID - just provide the full profile URL
- Supports various Instagram URL formats including mobile and desktop links
- Verification status helps identify authentic brand and creator accounts
