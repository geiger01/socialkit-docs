# Facebook Channel Stats API

The Facebook Channel Stats API retrieves comprehensive statistics and profile information from Facebook pages, including follower counts, verification status, profile images, and more.

## Endpoint

```bash copy
https://api.socialkit.dev/facebook/channel-stats
```

#### Example Request

```bash copy
GET https://api.socialkit.dev/facebook/channel-stats?access_key=<your-access-key>&url=https://www.facebook.com/mlb
```

## Parameters

**`url`** <small style={{margin:'0 10px'}}>string</small> <small style={{color: 'red'}}>Required</small><br/>
The Facebook page URL to retrieve statistics from.

---

**`access_key`** <small style={{margin:'0 10px'}}>string</small> <small style={{color: 'red'}}>Required</small><br/>
Your API access key. Can be provided via the `access_key` query parameter, `x-access-key` header, or request body.

---

**`cache`** <small style={{margin:'0 10px'}}>boolean</small> <small style={{margin:'0  10px  0 0'}}>Optional</small> <small>Defaults to false</small><br/>
Cache the response for faster subsequent requests.

---

**`cache_ttl`** <small style={{margin:'0 10px'}}>number</small> <small style={{margin:'0  10px  0 0'}}>Optional</small> <small>Defaults to 2592000</small><br/>
Cache the response for a custom TTL (in seconds). Maximum `2592000` seconds (1 month), minimum `3600` seconds (1 hour).

## Response Fields

- **`profileUrl`**: The complete URL to the Facebook page
- **`id`**: Unique identifier for the Facebook page
- **`fullName`**: Display name of the page
- **`bio`**: Page description/about section
- **`avatar`**: URL to the profile picture
- **`coverPhoto`**: URL to the cover photo
- **`verified`**: Whether the page is verified
- **`followers`**: Total number of followers
- **`category`**: Page category (e.g., "Sports league", "Public figure")
