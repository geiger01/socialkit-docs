# Screenshot API

The Screenshot API allows you to capture high-quality images of any webpage with a simple API request. You can customize the screenshot with various parameters like full-page capture, viewport size, and more.

## Endpoint

```bash copy
https://api.socialkit.dev/capture
```

#### Example Request

```bash copy
GET https://api.socialkit.dev/capture?access_key=<your-access-key>&url=https://tailwindcss.com
```

## Parameters

**`url`** <small style={{margin:'0 10px'}}>string</small> <small style={{color: 'red'}}>Required</small><br/>
The URL of the webpage to capture.

---

**`access_key`** <small style={{margin:'0 10px'}}>string</small> <small style={{color: 'red'}}>Required</small><br/>
Your API access key. Can be provided via the `access_key` query parameter, `x-access-key` header, or request body.

---

**`viewport_width`** <small style={{margin:'0 10px'}}>number</small> <small style={{margin:'0  10px  0 0'}}>Optional</small> <small>Defaults to 1280</small><br/>
The width of the browser viewport in pixels.

---

**`viewport_height`** <small style={{margin:'0 10px'}}>number</small> <small style={{margin:'0  10px  0 0'}}>Optional</small> <small>Defaults to 1024</small><br/>
The height of the browser viewport in pixels.

---

**`format`** <small style={{margin:'0 10px'}}>string</small> <small style={{margin:'0  10px  0 0'}}>Optional</small> <small>Defaults to 'png'</small><br/>
The output format of the screenshot (`webp`, `jpeg`, `jpg`, `png`, `pdf`).

---

**`device`** <small style={{margin:'0 10px'}}>string</small> <small style={{margin:'0  10px  0 0'}}>Optional</small> <br/>
The device type to emulate for the screenshot. <br/>
Available devices:

`iphone_14_pro_max`
`iphone_14_pro`
`iphone_13_pro_max`
`iphone_13_mini`
`galaxy_s23_ultra`
`galaxy_s23`
`galaxy_fold4`
`pixel_7_pro`
`pixel_6a`
`redmi_note_12_pro`
`redmi_note_11`
`huawei_p60_pro`
`huawei_mate_50_pro`
`iphone_x`
`iphone_12`
`pixel_5`
`galaxy_s8`
`ipad`

---

**`cache`** <small style={{margin:'0 10px'}}>boolean</small> <small style={{margin:'0  10px  0 0'}}>Optional</small> <small>Defaults to false</small><br/>
Cache the response.

---

**`cache_ttl`** <small style={{margin:'0 10px'}}>number</small> <small style={{margin:'0  10px  0 0'}}>Optional</small> <small>Defaults to 2592000</small><br/>
Cache the response for a custom TTL (in seconds). Maximum `2592000` seconds (1 month), minimum `3600` seconds (1 hour).

---

**`full_page`** <small style={{margin:'0 10px'}}>boolean</small> <small style={{margin:'0  10px  0 0'}}>Optional</small> <small>Defaults to false</small><br/>
Capture the entire page instead of just the visible viewport.

---

**`full_page_scroll`** <small style={{margin:'0 10px'}}>boolean</small> <small style={{margin:'0  10px  0 0'}}>Optional</small> <small>Defaults to true</small><br/>
Scroll the page to fully load lazy-loaded elements before capturing a full-page screenshot.

---

**`full_page_scroll_duration`** <small style={{margin:'0 10px'}}>number</small> <small style={{margin:'0  10px  0 0'}}>Optional</small> <small>Defaults to 400</small><br/>
Time in milliseconds for scrolling before capturing the full page.

---

**`scale_factor`** <small style={{margin:'0 10px'}}>number</small> <small style={{margin:'0  10px  0 0'}}>Optional</small> <small>Defaults to 1</small><br/>
The scale factor for high-resolution screenshots.

---

**`image_quality`** <small style={{margin:'0 10px'}}>number</small> <small style={{margin:'0  10px  0 0'}}>Optional</small> <small>Defaults to 80</small><br/>
Generate images with custom quality settings. <br/>
Supported formats include:

`jpeg`
`webp`

---

**`delay`** <small style={{margin:'0 10px'}}>number</small> <small style={{margin:'0  10px  0 0'}}>Optional</small> <small>Defaults to 0</small><br/>
Delay in seconds before capturing the screenshot (max `10s`).

---

**`wait_until`** <small style={{margin:'0 10px'}}>string</small> <small style={{margin:'0  10px  0 0'}}>Optional</small> <br/>
Define when to capture (`networkidle2`, `load`, `domcontentloaded`, `networkidle0`).

---

**`wait_for_selector`** <small style={{margin:'0 10px'}}>string</small> <small style={{margin:'0  10px  0 0'}}>Optional</small> <br/>
Wait for a specific element to appear before taking the screenshot.

---

**`selector`** <small style={{margin:'0 10px'}}>string</small> <small style={{margin:'0  10px  0 0'}}>Optional</small> <br/>
Capture a specific element on the page instead of the full viewport.

---

**`remove_selectors`** <small style={{margin:'0 10px'}}>string</small> <small style={{margin:'0  10px  0 0'}}>Optional</small> <br/>
A comma-separated list of elements to hide before capturing (e.g., ads, popups).

---

**`remove_cookie_banners`** <small style={{margin:'0 10px'}}>boolean</small> <small style={{margin:'0  10px  0 0'}}>Optional</small> <small>Defaults to false</small><br/>
Automatically remove cookie banners before capturing.

---

**`remove_ads`** <small style={{margin:'0 10px'}}>boolean</small> <small style={{margin:'0  10px  0 0'}}>Optional</small> <small>Defaults to false</small><br/>
Automatically remove ads before capturing.

---

**`block_urls`** <small style={{margin:'0 10px'}}>string</small> <small style={{margin:'0  10px  0 0'}}>Optional</small> <br/>
Comma-separated list of URL patterns to block (e.g., "analytics,tracking,advertisement"). You can specify URLs, domains, or simple patterns like "*.example.com/*".

---

**`block_resources`** <small style={{margin:'0 10px'}}>string</small> <small style={{margin:'0  10px  0 0'}}>Optional</small> <br/>
Comma-separated list of resource types to block (e.g., "image,stylesheet,font"). Available resource types:
`document`, `stylesheet`, `image`, `media`, `font`, `script`, `texttrack`, `xhr`, `fetch`, `eventsource`, `websocket`, `manifest`, `other`. Useful for optimizing page loading speed before capturing screenshots.

---

**`proxy`** <small style={{margin:'0 10px'}}>string</small> <small style={{margin:'0  10px  0 0'}}>Optional</small> <br/>
Specify a proxy server to route your request through. Supports HTTP, HTTPS, and SOCKS5 proxies. Format: `http://username:password@proxy.com:PORT`. Useful for bypassing geo-restrictions and rotating IPs.

---

**`s3_access_key_id`** <small style={{margin:'0 10px'}}>string</small> <small style={{margin:'0  10px  0 0'}}>Optional</small> <br/>
AWS S3 access key ID.

---

**`s3_secret_key`** <small style={{margin:'0 10px'}}>string</small> <small style={{margin:'0  10px  0 0'}}>Optional</small> <br/>
AWS S3 secret access key.

---

**`s3_bucket`** <small style={{margin:'0 10px'}}>string</small> <small style={{margin:'0  10px  0 0'}}>Optional</small> <br/>
The S3 bucket name where the screenshot will be uploaded.

---

**`s3_object_key`** <small style={{margin:'0 10px'}}>string</small> <small style={{margin:'0  10px  0 0'}}>Optional</small> <br/>
The filename for the S3 object (auto-generated if not provided).

---

**`storage_endpoint`** <small style={{margin:'0 10px'}}>string</small> <small style={{margin:'0  10px  0 0'}}>Optional</small> <br/>
Leave this field empty for AWS S3. Only specify a value if required. S3-compatible storage services are supportedâ€”for example, use `https://<accountId>.r2.cloudflarestorage.com` for Cloudflare R2.

---

**`s3_region`** <small style={{margin:'0 10px'}}>string</small> <small style={{margin:'0  10px  0 0'}}>Optional</small> <br/>
AWS S3 region for storage.
